valid方法：检查表单元素是否有效
rules()：获取表单元素的检效规则
rules()方法只针对表单的元素对象，而不是整个表单
例：获取id为username的input的rules:
$("#username").rules(); 输出为： Object { required: true, minlength: 2, maxlength: 10}
rules("add",reles);
例：添加一个规则
$("#username").rules("add",{email:true}); 输出为： Object {required: true, minlength: 2, maxlength: 10, email: true}
例：添加多个规则
$("#username").rules("add",{maxlenght:15,minlenght:1});
rules("remove",reles);
例：
$("#username").rules("remove","maxlength");
